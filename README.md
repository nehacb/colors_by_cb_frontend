# 🎨 Colors by CB - Art Marketplace (Frontend)

An enterprise-style Art Marketplace frontend built using **Angular**. The application provides a modern shopping experience with secure authentication, role-based access control, and responsive user interfaces.

## 🚀 Tech Stack

- Angular
- TypeScript
- RxJS
- Angular Router
- Reactive Forms
- HTTP Interceptors
- Route Guards
- JWT Authentication
- HTML5
- CSS3

---

## ✨ Features

### Authentication
- JWT Login
- Route Guards
- Role-Based UI
- HTTP Interceptor
- Persistent Login

### Products
- Product Listing
- Product Details
- Product Search
- Category Filter
- Create Product (Admin)
- Edit Product (Admin)
- Delete Product (Admin)

### Shopping
- Shopping Cart
- Cart Count
- Responsive Navbar

---

```
Angular Components

        ▼
```
